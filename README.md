## Hi! Welcome.
In this project I used:
- ReactJS as default framework.
- Material UI
- Localstorage

#How to use?
- After application running
- Click Add New User
- Fill fields
- Click to add.
- If you need EDIT, You have to click on PEN icon
- If you need DELETE, You need to click on TRASH icon

#How to run?
- Clone this project;
- run npm install
- after, npm start
